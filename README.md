#### PUT `/game`
##### Request:
```json
{
 "team1": 2,
 "team2": 3
}
```
##### Response:
```json
{
 "gameId": "3f4e"
}
```

#### POST `/game/{game_id}/score`
##### Request:
```json
{
 "team1": 2,
 "team2": 3
}
```
##### Response:
```json
{
 "team1": 2,
 "team2": 3
}
```

#### PUT `/game/{game_id}/start`
##### Response:
```json
{
 "gameOn": "True"
}
```

#### PUT `/game/{game_id}/stop`
##### Response:
```json
{
 "gameOn": "False"
}
```

#### POST `/game/{game_id}/time`
##### Response:
```json
{
 "gameTime": 120
}
```

#### POST `/game/{game_id}/teams
##### Request example:
```json
{
 "team1": 2,
 "team2": 3
}
```
##### Response:
```json
{
 "team1": 2,
 "team2": 3
}
```

#### PUT `/game/{game_id}/pause`
Works like a switch.
##### Response:
```json
{
 "gameOn": "True"
}
```
